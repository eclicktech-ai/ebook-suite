# 定位与边界

**ebook-suite** 用于存放个人或团队整理的电子书与相关资源（含铃声等非图书文件），与任何业务产品代码**无依赖**，可单独 clone、单独维护。

## 本仓库是什么 / 不是什么

| 说明 |
|------|
| **是**：主题书单（`library/expert/`）、个人书库（`library/personal/`）、媒体文件（`media/`）及说明文档。 |
| **不是**：后端、前端或 CI 构建的一部分。 |
