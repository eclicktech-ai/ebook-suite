# 定位与边界

**ebook-suite** 用于存放个人或团队整理的电子书与相关资源（含铃声等非图书文件），由多个历史 Git 仓库合并而来，**不保留**原仓库的 Git 历史。

## 本仓库是什么 / 不是什么

| 说明 |
|------|
| **是**：主题书单（`library/expert/`）、个人书库（`library/personal/`）、媒体备份（`media/`）的统一目录与说明文档。 |
| **不是**：任何产品后端、前端或构建流水线的一部分；可 **单独 clone、单独推送** 到 `eclicktech-ai/ebook-suite`。 |
